# Programming Fundamentals Assignment 1

This is the first assignment for the Programming Fundamentals course (CSWB120).

## Files

- `asgn1_mendoza.html` - HTML file demonstrating fundamental web technologies

## Description

This assignment explores the key differences between HTML, CSS, and programming. The page includes:

- **HTML & CSS Overview**: Explanation of how HTML provides structure and CSS handles presentation
- **Programming Constructs**: Introduction to three main types of programming statements:
  - Variables
  - IF statements
  - LOOPS statements
- **Interactive Button**: A clickable button that displays an alert message

## Features

- Styled webpage with blue background and yellow text
- Interactive JavaScript button that triggers an alert
- Responsive list of programming fundamentals

## How to Use

1. Open `asgn1_mendoza.html` in a web browser
2. Click the "Click Me" button to see the alert message
3. Review the content about HTML, CSS, and programming constructs
